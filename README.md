# racer
GUI Tool For HTTP Request Racing &amp; Request Dosing

# Setup & Usage
### $ git clone https://github.com/m00nisSmiling/racer.git
### $ pip install requests
### $ pip install termcolor
### $ cd racer
### $ chmod +x racer.py
### $ ./racer.py
